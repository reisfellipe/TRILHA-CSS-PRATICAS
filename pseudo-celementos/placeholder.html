Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de decrarala-lo no HTML, ou para selecionar uma parte de algum element.

identificado por ::palavra-chave

Pseudo-elemento - ::placeholder

Um placeholder é uma dica ara ajudar no preenchimento de algum campo de formulário

Aceita propriedades como:

  • color
  • font
  • background
  • text

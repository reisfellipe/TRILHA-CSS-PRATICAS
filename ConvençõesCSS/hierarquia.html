Hierarquia do Efeito Cascata

Estabeleceuma prioriade para aplicação de uma regra de estilo, caso se tenha mais de uma regra para um mesmo seletor CSS.

Se houver um conflito, o navegador irá

  1. Analisar todas as regras;
  2. Ver o nível de importância na rigem do elemento
  3. Comparar a especificidade das declarações com mesmo nível de importância (entra o cálculi de especificidade)
  4. Em caso de empate do grau de especificidade, irá olhar a ordem de declarações colocada dentro dos aquviso de código. A regra CSS que for declarada por último, irá ser selecionada.
